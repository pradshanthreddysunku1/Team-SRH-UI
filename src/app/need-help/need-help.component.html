Help:
This is a Language Translator website which can be used to translate from one language to another language.
A user will be able to select their desired language and select the language the user wish to translate to.
When the user enters a sentence or word, it will be translated to the desired language.
Our application has text to text language translation.
Our application has speech to text translation.

How to translate text:

  • Enter the text you want to translate in the box on the left.
  • Verify that the language of your content is correct.
  • Choose the language you want to translate to on the right.
  • The translated text will appear in the box on the right.
  • The input text field does not enable to enter your text until the input and output languages are selected.
